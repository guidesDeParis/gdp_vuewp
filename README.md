# Les Guides de Paris

# How to use
## clone this repos
## install
```
npm install
```
## run dev
```
npm run dev
```
## run prod
```
npm run prod
```

Based on VueJs template using Webpack 4 from the 'Vue.js and Webpack 4 From Scratch' article series on [itnext.io](https://itnext.io) from Daniel Cook.
